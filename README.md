# Commands

### Build
Ember (*.emb | *.mbr): 
```bash
$ pyro build sourceFile.mbr
```
Blaze (*.blz):
```bash
$ pyro build sourceFile.blz
```
### Run
Ember: // may not exist
```bash
$ pyro run sourceFile.mbr
```

Blaze:
```bash
$ pyro run sourceFile.blz
```

### Test
Ember:
```bash
$ pyro test sourceFile.mbr
```

Blaze:
```bash
$ pyro test sourceFile.blz
```
